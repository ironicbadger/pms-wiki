## Notes

Alex is joined by [Gardiner Bryant](https://gardinerbryant.com/) for a wider conversation about Jellyfin, Plex, and where open source media servers fit today.

The discussion is a pragmatic one. Jellyfin gives you control over your own media, but it still has rough edges. Client apps, plugins, music, live TV, file naming, scraping, and the overall user experience all come up as places where the project can still improve.

Jellyfin is not a perfect Plex replacement for everyone, but it remains worth paying attention to because it is open source, community-driven, and centered on the user's own library.
