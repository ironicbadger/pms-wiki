## Notes

Jellyfin 12.0 RC1 is a large release covering version numbering, performance work, database and library cleanup, TV episode version handling, FFmpeg 8.1 transcoding updates, HLS and Dolby Vision improvements, and web interface changes.

The episode also covers the upgrade side: backups, plugin compatibility, full library scans, API changes, OpenLibrary book metadata, audiobook chapters, and why most people should wait before moving their main Jellyfin server to 12.0 RC1.
